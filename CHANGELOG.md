## [1.0.1] - 2026-02-01

### Fixed

- Test release to verify release script functionality

## [1.0.0] - 2026-02-01

### Added

- Initial release of the TYPO3 Core Contribution Starter Kit
- MkDocs Material documentation site with 10 pages:
  - Home, Prerequisites, Accounts, Clone & Configure, DDEV Setup
  - TYPO3 Install, Contribution Workflow, Review Process
  - Troubleshooting, Command Reference
- Interactive setup script (`install.sh`) for one-liner configuration
- Config bundle with DDEV template, git hooks, and commit message template
- GitHub Actions workflow for automatic deployment to GitHub Pages
- Trademark notice and credits to TYPO3 Documentation Team
